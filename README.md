# 5in5 Live Site

This site is ready to be published with GitHub Pages.

## To deploy:
1. Commit and push your code to GitHub.
2. In your repo settings, set GitHub Pages to serve from the root (`/`) or `/docs` if you move your site there.
3. Optionally, run `npm run deploy` if you want to use the `gh-pages` branch for deployment.

## Local preview
Run:
```
npm install
npm start
```
Then open http://localhost:3000

---
- All your HTML, JS, and assets should be in the root or `/docs` folder.
- Make sure your asset paths are relative (not absolute) for GitHub Pages.
